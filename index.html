Eğer bu bilgisayarı ağdaki diğer istemcilerin bağlanacağı **Ana Sunucu** yapacaksanız, aşağıdaki ağ adreslerini kopyalayıp diğer bilgisayarlardaki istemci ayarlarına girmelisiniz.

Entegre REST API Sunucusu: ⏹ Durduruldu

Sunucu Kurulum Talimatları

  1. Yerel Kullanım:

    Uygulamayı tek bir bilgisayarda kullanıyorsanız bağlantı modunu "Yerel Mod" olarak bırakın. Veritabanınız bilgisayarınızda yerel olarak saklanır.

  2. Ortak Ağ Paylaşımı (Ağ SQL):

    Birden fazla bilgisayarın aynı veritabanına doğrudan bağlanması için, veritabanı dosyasını (`satis_takip.db`) ortak bir ağ klasörüne yerleştirin. Diğer bilgisayarlardan "Bağlantı Ayarları" -> "Ağ Paylaşım Modu" seçip ağ yolunu (örn: `\\SUNUCU-PC\Paylasim\satis_takip.db`) yazın.

  3. HTTP Server/Client Kurulumu (Gelişmiş):

    Uygulamayı tam bir istemci-sunucu mimarisine çevirmek için:

    • Ana sunucu bilgisayarında "Entegre REST API Sunucusu"nu başlatın veya `server.js` dosyasını harici bir sunucuda çalıştırın.
    • Sunucunun IP adresini ("Bu Bilgisayarın Ağ Adresleri" listesinden) kopyalayın.
    • Diğer tüm istemci bilgisayarlarında: "Bağlantı Ayarları" -> "HTTP İstemci Modu" seçip bu adresi girin.

Bağlantı Ayarları

Dilediğiniz bilgisayardan yerel klasör yolunu veya ağ klasör yolunu girin.
Bağlanmak istediğiniz sunucunun IP adresini girin (Örn: http://192.168.1.100:8765).
💡 Ağ Bağlantısı İpuçları (Sunucu & İstemci):
  • Yerel Yol Hatası: İstemci bilgisayarda `C:\` gibi yerel yollar girilirse sunucuya bağlanamaz. Bu yollar sadece o bilgisayarda geçerlidir.
  • Ağ Paylaşımı (UNC): Ağ Paylaşım modunda sunucu bilgisayardaki klasörü paylaşıma açmalı ve `\\SUNUCU-PC\Paylasim\satis_takip.db` şeklinde UNC ağ yolu girmelisiniz.
  • HTTP Sunucu (Önerilen): Router/Ağ üzerinden en sorunsuz bağlantı için sunucu bilgisayarda "Entegre API Sunucusunu" başlatın. İstemci bilgisayarlarda "HTTP İstemci Modu" seçip sunucu IP adresini (örn: `http://192.168.1.100:8765`) girin.
  • Güvenlik Duvarı Engeli: Bağlantı kurulamazsa, sunucu bilgisayarda **Windows Defender Güvenlik Duvarı** ayarlarına girip **8765** portu için "Gelen Kuralları"ndan yeni bir izin kuralı oluşturun.
